## 🎯 About

A modern, responsive portfolio website built from scratch using HTML and CSS. The design features a dark theme with gold accents, smooth animations, and a clean layout to present my professional profile effectively.

## ⭐ Features

- ✅ **Fully Responsive Design** - Works perfectly on all devices (desktop, tablet, mobile)
- ✅ **Smooth Navigation** - Fixed navbar with smooth scrolling to sections
- ✅ **Typing Animation** - Dynamic text effect in hero section
- ✅ **Hover Effects** - Interactive cards and buttons with gold accents
- ✅ **Social Media Integration** - Links to GitHub, LinkedIn, YouTube
- ✅ **Downloadable Resume** - Direct PDF download button
- ✅ **Clean UI/UX** - Professional dark theme with gold highlights

## 🛠️ Technologies Used

- **HTML5** - Structure and content
- **CSS3** - Styling, animations, and responsive design
- **Font Awesome** - Social media and section icons
- **Google Fonts** - Poppins font family

## 📑 Sections

The portfolio includes the following sections:

1. **Hero Section** - Introduction with profile image and typing effect
2. **About Me** - Brief professional summary
3. **Education** - Academic background with CGPA
4. **Skills** - Categorized skills with interactive tags
5. **Interests** - Professional interests and focus areas
6. **Projects** - Featured academic and personal projects
7. **Achievements** - Academic and project accomplishments
8. **Languages** - Language proficiency
9. **Contact** - Email and phone information
10. **Feedback** - Testimonials from faculty and peers
